Clustering des offres

Ce modèle de clustering K-means segmente les offres d'emploi en exploitant les variables latentes extraites via une matrice TF-IDF des descriptions textuelles, assurant une structuration thématique fine du corpus. Le modèle est entraîné sur les 50 premières composantes issues d’une SVD afin de réduire le bruit et optimiser l’apprentissage.

Projections sur le plan

Les clusters sont ensuite projetés en 2D ou 3D au moyen d’un t-SNE, méthode de réduction de dimension conservant les relations de voisinage pour une représentation factorielle interprétable.
